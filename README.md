## Requirements

You must have Node.js and npm installed on your machine. This project was built against the following versions:
 
- Node v16.3.0
- npm v7.15.1

## Setup

- Clone the repository.
- Run following commands in the project directory to start.

```bash
npm install
npm run dev
```
